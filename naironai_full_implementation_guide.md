### Rive

**Core Purpose:** Rive is a real-time interactive design and animation tool used to create and embed vector animations and interactive motion graphics into applications and websites. It allows developers and designers to integrate high-performance, state-driven animations that can respond to user inputs and app logic.

**Tech Stack:** Rive supports multiple platforms and integrates with popular tech stacks, notably: React (via rive-react), Flutter, iOS (Swift), Android (Kotlin/Java), and Web (JavaScript/TypeScript). The primary usage in web/front-end projects typically involves React or vanilla JavaScript environments.

**Installation:**
```bash
npm install rive-js
# For React specifically:
npm install rive-react
```

**Setup:** {'Key files to create or modify': ['Add/import the Rive animation file (.riv) to your project assets folder.', 'In a React project, create a component file (e.g., RiveAnimation.jsx or RiveAnimation.tsx) to encapsulate the Rive animation logic.', 'No additional config files like tailwind.config.js are required specifically for Rive, but ensure your bundler handles importing static .riv files or serve them from a public path.', 'Optionally, configure TypeScript typings or create a utils file to wrap common Rive animation controls for reuse.'], 'Example modification': "In your react component or wherever you manage UI animations, import and use the <Rive> component from 'rive-react'."}

**Usage Example:**
```tsx
import React from 'react';
import { Rive } from 'rive-react';

const MyRiveAnimation = () => {
  return (
    <div style={{ width: '300px', height: '300px' }}>
      <Rive 
        src="/assets/animations/example.riv"

        stateMachines={["State Machine 1"]}
```

```bash
['npm install lucide-react framer-motion', 'or', 'yarn add lucide-react framer-motion']
```
